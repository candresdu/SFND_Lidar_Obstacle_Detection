## Local Installation

### Ubuntu 

1. Execute the following commands in a terminal

   ```shell
   cd ~/submission
   mkdir build && cd build
   cmake ..
   make
   ./submission
   ```

   This should run the clustering and plane segmentation code on dataset under data/pcd/data_1
